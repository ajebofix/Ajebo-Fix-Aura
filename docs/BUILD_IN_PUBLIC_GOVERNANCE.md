# Aura Build-in-Public Governance

Owner: Ajebo Fix Ltd  
Status: Active internal engineering rule  
Last reviewed: 2026-07-20

## Core rule

Build trust in public. Build competitive machinery in private.

## Public-safe material

- Founder story and product mission
- Completed milestones
- Non-confidential interface previews
- Anonymised or fictional demo records
- Released capabilities
- General automotive-health education

## Review-required material

- Product screenshots
- Technical descriptions
- Future features
- Metrics, pricing or partnerships
- AI, diagnostics or predicted-outcome claims

## Never publish

- Source code excerpts for publicity
- Secrets, API keys or credentials
- Production customer records
- Real VINs, plates, phone numbers or service histories
- Database diagrams, scoring formulas or private prompts
- Security architecture
- Patent-sensitive technical mechanisms
- Detailed unreleased roadmaps

## Demo-data requirement

All public product demonstrations must use clearly fictional demo accounts and vehicle records. Screenshots must be reviewed and redacted before publication.

## Product-claim rule

Aura must not describe monitoring as diagnosis, guidance as guaranteed repair advice, estimates as confirmed outcomes, AI output as mechanic-verified findings, or prototypes as completed/live products.

## Brand asset roles

- Ajebo Fix primary logo: parent/company brand
- Aura shield logo: main Aura product identity
- Aura health mini logo: compact health/status UI mark only

The mini mark must not replace the main Aura identity in legal, trademark or principal product-brand use.

## Public narrative

> Ajebo Fix is building the private automotive-health infrastructure that helps serious vehicle owners understand, manage and act on their vehicle’s condition before problems become expensive.

## Approval record

The authoritative content-approval register, legal templates, invention disclosures and confidential-information register are maintained in the company Google Drive, not in this public repository.
